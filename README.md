# GPB Special

**Симуляция работы с реальными данными с использованием необходимых в работе инструментов**
Стэк: Python, SQL, Groq API, Streamlit, LightGBM, Pandas, Sqlite3
Задачи проекта:
1) Генерация данных: имитируем корсчет
	- несколько типов транзакций
	- для каждого типа транзакций случайное нормально распределенное значение
2) SQL исследование
3) ML
	- генерация фичей из исходного датасета для модели
	- построение бейзлайна
	- улучшение
	- объяснимость 
	- метрики: MAE, R2
4) BI
5) Few-Shot: веб интерфейс, с выводом результата
